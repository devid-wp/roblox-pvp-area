# UI и Анимации — Sekiro-Style

## Расположение UI
Весь HUD в **левом верхнем углу** экрана (отступ 20px).

## Элементы UI

### 1. Имя игрока
- Жирный шрифт, цвет тёплого золота (#DCC8A0)
- Размер: 18pt

### 2. Полоска HP (здоровье)
- **Ширина**: 320px, **Высота**: 28px
- **Фон**: тёмно-красный (#281414)
- **Заполнение**: красный градиент сверху вниз (#FF7850 → #A01E1E)
- **Особенности**:
  - Плавное уменьшение при получении урона
  - При HP < 30%: мигание красной обводки
  - Текст показывает "100 / 100"

### 3. Полоска POSTURE (стойка) — фишка Sekiro
- **Ширина**: 320px, **Высота**: 16px
- **Фон**: тёмно-фиолетовый (#1E1428)
- **Заполнение**: фиолетовый градиент (#DC8CFF → #783CB4)
- **Особенности**:
  - Заполняется при получении урона
  - При 100% — оглушение 2.5 сек
  - Метка "POSTURE" слева

### 4. Индикатор руко-хвата (Q)
- **Ширина**: 320px, **Высота**: 50px
- **Иконка Q**: круглая кнопка 36x36px с буквой "Q"
- **Обводка**: меняет цвет в зависимости от состояния
- **Текст статуса**:
  - "● ГОТОВ" (зелёный #B4DC82) — можно использовать
  - "● ПЕРЕЗАРЯДКА" (оранжевый #DC9650) — на КД
  - В воздухе: мигание красной обводки

### Кулдаун руко-хвата
- **Базовый КД**: 4 секунды
- **В воздухе**: +2 секунды
- **Визуал**: 
  - Тёмный оверлей "съедает" иконку снизу вверх
  - Число в центре показывает оставшееся время
  - При готовности оверлей исчезает с плавной анимацией

## Анимации

### Атаки катаной
- `KatanaLightAttack1-4` — комбо из 4 ударов
  - Каждый следующий удар чуть быстрее (+5% speed)
  - FadeTime: 0.1 сек между ударами
  - Priority: Action

### Парирование
- `KatanaParry` — короткая анимация 0.3 сек
- **Успешное парирование** (Speed 1.2):
  - 5 искр разлетаются от игрока
  - Звук "clang" (TODO)
  - Сброс Posture

### Блок
- `KatanaBlock` — зацикленная анимация
- Воспроизводится пока зажата клавиша блока

### Оглушение
- `KatanaStunned` — длится 2.5 сек
- Над головой: красный значок 💫, качающийся из стороны в сторону
- Игрок не может атаковать/блокировать

### Руко-хват
- `HookFire` — анимация выстрела 0.4 сек
- `HookPull` — зацикленная анимация притягивания
- **Визуал цепи**: тонкая Part от игрока к точке захвата, исчезает за 0.3 сек

### Движение
- `DoubleJump` — анимация второго прыжка
- `AirDash` — рывок в воздухе
- `WallRun` — бег по стене (1.5 сек)

### Сюрикены
- `ThrowShuriken` — бросок
- Визуал: маленький металлический диск, летит и вращается

## Эффекты (Tween-анимации)

| Эффект | Описание | Длительность |
|--------|----------|--------------|
| SlashEffect | Линия разреза при ударе | 0.2 сек |
| ParrySpark | Искры при парировании | 0.4 сек |
| HookChain | Цепь руко-хвата | 0.3 сек |
| LandingDust | Пыль при приземлении | 0.6 сек |
| StunIcon | Значок оглушения | 2.5 сек |
