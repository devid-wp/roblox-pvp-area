# Способности/Перки

| Способность   | Описание                            | Кулдаун |
|---------------|-------------------------------------|---------|
| Dash (Рывок)  | Мгновенный рывок вперёд             | 5 сек   |
| Heal (Хил)    | Восстанавливает 30 HP               | 15 сек  |
| Double Jump   | Дополнительный прыжок в воздухе     | 0 сек   |
| Shield (Щит)  | Поглощает 50% урона на 3 секунды    | 20 сек  |

## Балансировка
- На матч игрок выбирает 2 способности из 4
- Подбираются способности из разных категорий (Movement/Defense/Support) для разнообразия
